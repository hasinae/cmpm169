Experiment 4: Images, Video, & Sound Art

Description

What are you trying to accomplish with this experiment?

In this experiment, I aimed to create a glitch-inspired visual effect using a live camera feed. The goal was to manipulate the video's pixel data in real-time to produce dynamic distortions and color shifts, resulting in a visually engaging and surreal experience. By incorporating sine waves and random pixel manipulations, I sought to achieve a glitchy, animated transformation of the camera feed.

Technical

Describe in detail what you did technically and how it works?

Technically, I used p5.js to capture and process the live camera feed. The `createCapture(VIDEO)` function initializes the video feed, and `video.loadPixels()` allows access to the pixel data for manipulation. I applied distortions by shifting pixel values dynamically using sine waves and random offsets. The `video.updatePixels()` function updates the video with the modified pixel data, creating the glitch effect.

To enhance the visual impact, I incorporated color cycling using the HSB color mode, which allowed for vibrant and smooth color transitions. The sine wave distortions were controlled by parameters such as amplitude and frequency, which determined the intensity and rhythm of the glitches. The final output is displayed on the canvas using the `image()` function, creating a real-time, glitchy transformation of the camera feed.

Reflection

Each person that worked on the code should reflect on the process, the difficulties, and the successes of the experiment.

Reflection: The process involved experimenting with p5.js to create a glitch-inspired visual effect using a live camera feed. Initially, I encountered challenges in manipulating the video's pixel data and ensuring smooth distortions. There were a lot of errors related to uninitialized objects and incorrect function calls which needed a lot of debugging.

By adjusting parameters and incorporating sine waves for dynamic effects, I achieved a more reliable result. The distortions became more fluid, and the overall visual output felt more polished and aligned with my vision. This step required extensive testing to balance the intensity of the glitches.

The final outcome is a glitchy, animated transformation of the camera feed, creating an engaging and visually dynamic experience. This project allowed me to explore real-time video processing and creative coding techniques effectively, demonstrating the potential of combining technical and artistic approaches in code.
