Experiment 5: 3D Graphics

Description

What are you trying to accomplish with this experiment?

In this experiment, I aim to create an interactive 3D physics sandbox where objects respond to user input and simulate realistic behaviors such as gravity, bouncing, and collisions. Additionally, I want to incorporate creative features like explosions triggered by double-clicks and dynamic color changes based on the velocity of the objects. The goal is to combine physics principles with interactive and visually engaging elements, creating a playful and dynamic environment for users to explore.

Technical

Describe in detail what you did technically and how it works?

I created an interactive physics sandbox where balls respond to gravity, user inputs, and collisions. The balls were programmed to stay within the sandbox by bouncing off the edges. I added an explosion effect that triggers on double click, causing nearby balls to scatter outward. Additionally, the balls change color based on their speed, adding a visual representation of their motion and making the sandbox more engaging. This project combines physics simulation with creative interactivity to deliver a dynamic and playful experience.

Reflection

Each person that worked on the code should reflect on the process, the difficulties, and the successes of the experiment.

Reflection: The process of creating the interactive physics sandbox in p5.js was both challenging and rewarding. My goal was to simulate a 3D physics playground/sandbox while also incorporating creative features like explosions and dynamic color changes based on velocity. I came across difficulties with the double clicking feature, but I came across them quickly when I looked at my code more closely. Adjusting the physics mechanics, such as decreasing velocity during bounces, required a lot of experimentation. Incorporating the explosion effect on double click added layers of interactivity. These features made the sandbox feel a lot more engaging. The final result is an interactive 3D physics playground, where the objects respond to the user's actions. This project taught me the importance of blending technical elements with creative elements to create an really cool and engaging user experience. It deepened my understanding of 3D physics simulations and allowed me to learn more about the importance of experimentation and creative freedom in coding.
